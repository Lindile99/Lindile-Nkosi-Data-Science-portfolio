## 🏥 Healthcare Admissions Analysis (SQL + Excel)

Analyzed **55,000+ hospital admission records** using SQL and built an **interactive Excel dashboard** to explore admission trends, insurance revenue, and patient demographics.

**Tools:**  
SQL (MySQL), Excel (Pivot Tables, Charts, Slicers)

**Key Insights**

- Analyzed **monthly hospital admission trends**
- Identified **revenue generated by different insurance providers**
- Examined **average hospital stay by admission type**

## 🩺 Health Insurance Claim Fraud Detection (Machine Learning)

Developed a **machine learning model** to detect fraudulent health insurance claims using **Random Forest Classification**, with Logistic Regression used as a benchmark model.

**Tools:**  
Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn

**Key Insights**

- Cleaned and preprocessed insurance claim data from **Zenodo**
- Performed **exploratory data analysis to identify fraud patterns**
- Applied **Random Forest classification for fraud detection**
- Evaluated model performance using **Accuracy, Precision, Recall, and F1 Score**
- Addressed class imbalance using **SMOTE**

🔗 **Project Repository**  
https://github.com/Lindile99/Health-Insurance-Claim-Fraud-Detection
